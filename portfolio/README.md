# Arvin John C. Sison - Portfolio

A personal portfolio website showcasing my educational background, certifications, and IT projects.

## Features
- Responsive design
- Interactive certificate viewer
- Project carousel
- Modal image viewer
- Modern gradient design

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## Deployment
This portfolio is ready for deployment on Vercel, Netlify, or GitHub Pages.

### Deploy to Vercel
1. Push to GitHub repository
2. Connect repository to Vercel
3. Deploy automatically

### Local Development
Simply open `public/index.html` in your browser.
